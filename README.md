## 🚀 Led
### **[SmartGallery](https://github.com/FSU-CloudClub/CloudClub-Spring26-ImageManagementWebApp)**
A serverless image recognition and management app built to mirror a professional production environment. Users upload images that are auto-tagged via Amazon Rekognition, with tags stored in DynamoDB for browsing, filtering, and search. Built with IAM least-privilege principles, private S3 buckets, and authenticated access throughout.
